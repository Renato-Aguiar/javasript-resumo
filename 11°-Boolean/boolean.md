# operadores de comparação
* == (igualdade)
* !=  (desigualdade)
* '>   (maior que)'
* '>= (maior ou igual)'
* <   (menor)
* <= (menor ou igual)

### este operador é usado para saber se dois valores são iguais. Se for, o javascript responde com true e se não forem, com false.
